## YAML PARSER

Приложение принимает один файл openapi.yaml, парсит его и делит на части components и paths.

Установка и запуск
1. Клонируйте репозиторий
```bat
git clone https://github.com/ianmoore228/kode-test.git
cd kode-test
```
2. Установите зависимости
```bat
npm install
```
3. Запустите проект
```bat
npm run dev
```

Или проверьте тут:
